# Richiamare altri Makefile

Posso fare dei target che richiamano altri makefile sfruttando la possibilit√†
di fare dei comandi:

```Makefile
grafo.o:
	cd ../grafo
	make depends
    make
    cp grafo.o $(PROJ_DIR)
    cd $(PROJ_DIR)
```

altrimenti posso anche invocare Make da quella cartella direttamente nella mia:

```Makefile
grafo.o:
    make -f ../grafo/Makefile
```
